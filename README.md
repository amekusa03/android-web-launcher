# らん茶 (WebLauncher)

らん茶は、Androidのホーム画面に特定のWebサイトを開くためのショートカット（スロット）を最大5つまで追加できるアプリです。
アイコンは静岡生まれ「お茶缶（円柱）」のデザインを採用しています。

## 特徴

- **マルチスロット構成**: 最大5つの独立したランチャーアイコン（Web①〜Web⑤）を利用可能。
- **お茶缶デザイン**: 各スロットとアプリ本体に、色分けされた円柱型（お茶缶）アイコンを採用。
- **透過型ランチャー**: `LauncherActivity` は透過テーマを採用しており、ユーザー体験を損なわずにバックグラウンドでリダイレクト処理を行います。
- **動的なアイコン管理**: 各スロットは `activity-alias` として定義されており、プログラム側から有効/無効を切り替えることができます。
- **簡単カラーカスタマイズ**: 16進数を知らなくても、カラーパネルから直感的にアイコン色を選べます。

## プロジェクト構成

### 画面（Activity）

1. **管理画面 (`MainActivity`)**:
    - アプリの設定や各スロットの有効化・無効化、URLの割り当てを行うためのメインUIです。
    - ランチャー上のラベルは「Webランチャー」として表示されます。

2. **実行ロジック (`LauncherActivity`)**:
    - 各スロットから起動される共通のActivityです。
    - `Theme.Translucent.NoTitleBar` を使用しており、画面自体は表示されず、目的のURLへ遷移するロジックを実行します。

### スロット (`activity-alias`)

以下の5つのエイリアスが定義されており、それぞれ異なる色のお茶缶アイコンが割り当てられています。

- **Web①**: 青
- **Web②**: 赤
- **Web③**: 緑
- **Web④**: 黄
- **Web⑤**: 紫

## 仕組み

1. 管理画面でスロットにURLを設定し保存する。
2. `PackageManager` を介して即座に `activity-alias` が有効化され、ドロワーにアイコンが出現する。
3. アイコンをタップすると、設定されたURLが Chrome Custom Tabs またはブラウザで開く。
4. 「ホーム画面に追加」機能により、ユーザー指定のラベル名でショートカットを作成可能。

## 対象外

設定された名前をドロワーに表示されることはできませんでした。
Nova Launcherでは変更できる事、AQUOS HOMEでは変更できない事は確認しました
